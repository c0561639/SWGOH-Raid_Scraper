# SWGOH-Raid Scraper

## Overview
SWGOH-Raid Scraper is a Discord bot designed to track guild raids for Star Wars: Galaxy of Heroes.
It utilizes swgoh.gg to provide this information and help track member activity.

## Installation
1. Clone repo
```bash
git clone https://github.com/c0561639/SWGOH-Raid_Scraper.git
```
2. Open sln in Visual Studio
3. Create .env with DISCORD_WEBHOOK_URL
4. Build and run
